## üõ†Ô∏è How to Install (For Beginners!)

Since this is a custom extension, it is not in the Chrome Web Store. Don't worry, installing it takes exactly 1 minute!

### Step 1: Download the Code from GitHub
1. Scroll to the top of this GitHub page.
2. Look for the green button that says **"<> Code"** and click it.
3. At the bottom of the little menu that pops up, click **"Download ZIP"**.
4. The file will download to your computer (usually in your `Downloads` folder).

### Step 2: Extract (Unzip) the Downloaded File
*Chrome cannot read a zipped file directly, so we need to unzip it first!*
1. **Windows:** Find the downloaded `.zip` file, right-click it, and select **"Extract All..."**. Click "Extract". 
2. **Mac:** Find the downloaded `.zip` file and just double-click it. It will automatically extract into a normal folder.
3. You should now have a normal, unzipped folder containing files like `manifest.json`, `content.js`, etc. Remember where this folder is saved!

### Step 3: Open Chrome Extensions
1. Open Google Chrome.
2. Click the **Puzzle Piece icon** üß© in the top right corner of your browser (next to your profile picture).
3. Click **"Manage Extensions"** at the bottom of the menu. 
   *(Alternatively, type `chrome://extensions/` into your URL bar and hit Enter).*

### Step 4: Turn on Developer Mode
1. Look at the **top right corner** of the Extensions page.
2. You will see a toggle switch that says **"Developer mode"**. Turn it **ON**.

### Step 5: Load the Extension
1. Once Developer mode is on, three new buttons will appear on the top left of the screen.
2. Click the button that says **"Load unpacked"**.
3. A file browser window will pop up. Find and select the **extracted/unzipped folder** from Step 2. *(Make sure you select the unzipped folder, NOT the original .zip file!)*
4. Click **Select Folder**.

üéâ **Boom! You're done.** The extension is now installed.

## ‚å®Ô∏è Keyboard Shortcuts (Pro-Tips)

To make your mock test practice incredibly fast, use these keyboard shortcuts instead of your mouse:

* Press **`1`**, **`2`**, **`3`**, or **`4`** (or `A`, `B`, `C`, `D`) ‚ûî Instantly selects the option.
* Press **`Enter`** ‚ûî Submits the answer / Checks the solution.
* Press **`Right Arrow (‚Üí)`** ‚ûî Save & Next (Goes to the next question).
* Press **`Left Arrow (‚Üê)`** ‚ûî Goes to the previous question.

*Bonus Trick:* If you are at the bottom of a question page, simply scroll down using your mouse wheel/trackpad, and it will automatically trigger "Save & Next" for you!

## üß∞ Troubleshooting

* **The page looks weird or broken!**
  * Sometimes the page needs a fresh start. Click the extension icon and hit the big red **"WIPE ALL DATA & RELOAD"** button.
* **The shortcuts aren't working!**
  * Make sure you aren't currently clicked inside a text box or search bar. Click anywhere on the blank background of the page and try again.
* **I want to exit NTA mode.**
  * Just click the extension icon and turn OFF the "Enable Interface" toggle.
